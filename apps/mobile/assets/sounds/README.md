# EricPay sonic mark

`ericpay-received.wav` is an original, provisional sonic mark generated for this prototype from two synthesized tones and the macOS `Samantha` system voice saying “EricPay”. It does not reuse or sample Yape audio. The app loads it as a JavaScript asset with `expo-audio` while EricPay is open; it is not configured as an APNs or native push sound.

Before public distribution, replace the spoken layer with a professionally recorded or explicitly licensed voice while preserving the filename and a sub-second duration. Recommended delivery format: mono PCM WAV, 44.1 kHz, 16-bit, peak below -2 dBFS.
